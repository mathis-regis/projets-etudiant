**Guide de lancement des différents projet ainsi que résumé**
1)Projet du démineur:
*Résumé*
Ce projet étais celui du premier semestre de la L1 en programation impérative et consistais a crée un démineur, on peut poser des drapeaux sur les cases avec le clic droit de la souris(pour marquer les cases qu'ont suspecte être des bombes), découvrir une case avec le clic gauche (mais attention au bombes qui vous font perdre et recommencer la partie), les case avec bombes sont aléatoires pour chaque partie et quand on révéle une case sans bombes le nombre de bombe autour de cette case est indiqué.On a un compteur de drapeau restant en haut a droite et un chronométre des secondes passer sur une partie, pour commencer une nouvelle partie il faut cliquer sur le smiley en haut. On gagne quand toute les cases sans bombes sont révélés.
Ce projet a été réalisé a l'aide de processing donc il faut avoir processing d'installer sur la machine
*Comment Lancé*
-dézipper le fichier
-ouvrir le fichier avec processing "GMI4_REGIS_Mathis_Demineur_version_finale.pde", puis l'éxécuter.
